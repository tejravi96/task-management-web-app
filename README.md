# Task Management Web Application

A full-stack web application to manage daily tasks efficiently using Angular, Java, and MySQL. The application supports complete CRUD operations with seamless frontend-backend integration.

---

## 🚀 Tech Stack

* Frontend: Angular, HTML, CSS, JavaScript
* Backend: Java (Spring Boot)
* Database: MySQL
* API: RESTful APIs

---

## 📌 Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Simple and user-friendly interface

---

## 🏗️ Project Architecture

Frontend (Angular) → REST API → Backend (Java) → MySQL Database

## ⚙️ Setup Instructions

### 1. Clone Repository

git clone https://github.com/your-username/task-management-web-app.git
cd task-management-web-app

### 2. Setup Backend

* Open backend in IDE
* Configure MySQL in application.properties
* Run Spring Boot application

### 3. Setup Frontend

cd frontend
npm install
ng serve

### 4. Access Application

http://localhost:4200

---

## 🗄️ Database Setup

* Create database in MySQL
* Run schema.sql file
* Update DB credentials in backend
